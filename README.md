# 5In1RowService

一个基于python2.7、netsteam的联网五子棋服务端，使用TCP通信，实现登录、游戏大厅、聊天、排行榜、创建房间、加入房间、下棋、悔棋、禁手等功能

客户端地址：https://github.com/caiwb/5In1RowClient
